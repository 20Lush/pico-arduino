
## Overview

Here is a quick demo for the Pico ADC with the analogRead() functionality. 
On the Pico we can use the pins GP26 to GP28 for reading analog signals. 

## Connections 
To test the functionality I used a CJMCU-622 Microphone

 622    | Pico              
--------|------------------------
 VCC    | 3.3V (OUT) 
 GND    | GND 
 OUT    | GP25
